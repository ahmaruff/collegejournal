Blockquotes

untuk membuat blockquotes, setiap awal line tulisan perlu di tambah tanda '>'

Markdown

> ## blockquoted header
> blockquoted text 1
> blockquoted text 2
> blockquoted text 3

output

Blockquoted header

blockquoted text 1

blockquoted text 2

blockquoted text 3

Header

untuk membuat header/headings, setiap awal line tulisan perlu di tambah tanda '#'

Markdown

# Heading 1
## Heading 2
### Heading 3

output

Heading 1

Heading 2

Heading 3

New line/enter

untuk membuat new line/enter, setiap akhir line tulisan perlu di tambah 2 tanda spasi

Markdown

new line 1
new line 2

output

new line 1

new line 2

Block Code

untuk membuat block code, setiap awal line tulisan perlu di tambah indentasi berupa 4 spasi

Markdown

## headings 2
wwwblock code line 1
wwwblock code line 2
teks biasa

output

Headings 2

block code line 1
block code line 2

text biasa

Inline Code

untuk membuat Inline code, setiap awal & akhir tulisan perlu di tambah tanda (`)

Markdown

ini `inline code` bro

output

ini inline code bro

Italic dan Bold

untuk membuat tulisan italic, setiap awal & akhir tulisan perlu di tambah satu tanda (*) atau (_)

sedangkan untuk bold, setiap awala dan akhit tulisan perlu ditambah DUA tanda (*) atau (_)

Markdown

ini text *italic* ini juga _italic_
ini text **bold** ini juga __bold__

output

ini text italic ini juga italic
ini text bold ini juga bold

Horizontal Rule

untuk membuat Horizontal rule, pada line yang akan diberi garis bawah, beri tanda strip (-) 3 kali atau lebih

Markdown

random text
---

output

random text

inline Link

Markdown

this is an [example link](https://example.com)

output

this is an example link

link Reference

link dengan gaya referensi memudahkan kita untuk memanggil link itu berdasarkan nama. yang mana bisa kita panggil dibagian manapun pada page tersebut

Markdown

ini adalah panduan dari [Markdown][1]
[1]: https://markdownguide.org "Markdown"

output

ini adalah panduan dari Markdown

- list

+ list
